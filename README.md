# Dane & Emmy's Gameshow Quiz

## Project Description
This is a gameshow about Dane and Emmy's fandoms and interests that cures boredom. The fandoms we used are DC, F1, and Valorant. It asks questions and gives points everytime you get a questions correct.

## Features
- Ability to choose a starting fandom (DC, F1, or Valorant) and answer randomized questions for each category.
- GUI-based quiz using Tkinter with background images for different screens.
- Multiple fandom categories with questions displayed in random order.
- Answer selection using buttons, with instant checking and immediate feedback (“Correct” or “Incorrect”).
- Score system where each correct answer gives 1 point, with tracking per fandom and overall performance.
- Ability to view final results when quitting, including total score and accuracy percentage.
- Option to replay the quiz after finishing.
- Admin login system for managing the quiz content.
- Admin panel that allows adding new questions and removing existing ones dynamically.
- Dropdown menu used for selecting and deleting specific questions.


## How to Run the Program
1. Make sure you have Python and VsCode installed.
2. Make sure you have the folder with the python code and correctly named pictures in png.
3. Open the folder in VsCode.
4. Click the run button(triangle)
5. Enjoy playing!
